# GuessNumber-python-project
GuessTheNumber / AdivinaElNúmero
🎮 Juego: Adivina el Número (4 dígitos)
📌 Descripción

Este proyecto consiste en un juego interactivo donde el usuario debe adivinar un número secreto de 4 dígitos generado aleatoriamente por el sistema.

El objetivo principal de este proyecto es practicar y fortalecer habilidades en programación con Python y desarrollo de interfaces gráficas utilizando Kivy.

🚀 Características
Generación aleatoria de un número de 4 dígitos
Interfaz gráfica amigable desarrollada con Kivy
Validación de entradas del usuario
Sistema de intentos para adivinar el número
Retroalimentación al usuario (pistas o mensajes)
🛠️ Tecnologías utilizadas
Python
Kivy (para la interfaz gráfica)
🎯 Objetivo del proyecto

Este proyecto fue desarrollado como una práctica personal para:

Mejorar habilidades en Python
Aprender a trabajar con interfaces gráficas usando Kivy
Comprender la lógica detrás de juegos simples
Desarrollar proyectos individuales desde cero
📂 Estructura del proyecto: 
-

Clona este repositorio:

git clone https://github.com/tu-usuario/adivina-numero.git

Instala las dependencias:

pip install kivy

🎮 Cómo jugar
Inicia el juego.
Ingresa un número de 4 dígitos.
El sistema te dará retroalimentación.
Sigue intentando hasta adivinar el número correcto.
